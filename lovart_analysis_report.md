## 二、核心功能与产品架构

### 2.1 功能矩阵

Lovart 覆盖了从品牌设计到营销素材的全链路：

| 功能模块 | 说明 |
|---------|------|
| **对话式设计生成** | 用自然语言描述需求，AI 自动生成专业设计 |
| **Logo 设计** | 品牌 Logo 自动生成，支持多风格多方案 |
| **产品页面设计** | 电商产品页、Hero 图、产品特写等 |
| **社媒素材** | Instagram/小红书/TikTok 等多尺寸适配 |
| **视频广告** | 营销视频、品牌宣传片生成 |
| **品牌系统** | 完整的品牌 VI 体系（配色、字体、调性） |
| **海报/宣传单** | 活动海报、促销传单等印刷品设计 |
| **包装设计** | 产品包装概念图生成 |

### 2.2 支持的 AI 模型引擎

Lovart 集成了多个底层模型，按场景智能调度：

| 模型 | 用途 |
|------|------|
| Flux | 图像生成（Stability AI 生态） |
| DALL-E | 图像生成（OpenAI） |
| Nano Banana | 轻量快速生成 |
| Sora 2 | 视频生成（OpenAI） |
| Veo 3.1 | 视频生成（Google DeepMind） |
| 自研模型 | 设计排版、布局优化等专项能力 |

> **核心差异点**：Lovart 不是单一模型工具，而是**模型编排层**——根据用户意图自动选择最合适的模型组合完成设计任务。

### 2.3 产品形态

| 形态 | 说明 |
|------|------|
| **Web App** | 浏览器即用，无需安装 |
| **对话式 Agent** | 非传统"模板编辑"，而是 AI Agent 交互 |
| **多语言支持** | 12+ 语言（英/中/日/韩/西/法/德/葡/俄/阿等） |

## 三、技术栈分析

### 3.1 前端架构

| 层级 | 技术选型 |
|------|---------|
| **框架** | Next.js（lovart.ai）、Astro（lovart.me） |
| **UI 库** | Mantine（基于 React 的组件库） |
| **样式** | CSS Variables、Mantine Theme System |
| **静态资源 CDN** | assets-persist.lovart.ai（独立图片域名） |
| **JS 产物** | 静态化 Chunk 部署（web-static3.lovart.ai） |

### 3.2 基础设施

| 服务 | 用途 |
|-----|------|
| **Cloudflare** | CDN 加速 + Turnstile 人机验证 |
| **Google Tag Manager** | 流量分析 & 事件追踪 |
| **Ptengine** | 用户行为热力图分析 |
| **Structured Data** | Schema.org 标准 SEO 标签 |

### 3.3 技术亮点

- **SSR + 静态化混合**：Next.js 官网使用 SSR，产品页面使用 Astro 静态生成
- **分域名部署**：官网（lovart.ai）与应用（lovart.me）物理分离
- **图片资产独立**：专用资产域名减少主站负载
- **多语言 i18n**：built-in 支持 12 种语言，布局适配 RTL

## 四、商业模式与定价

### 4.1 定价模型

| 方案 | 价格 | 核心权益 |
|------|------|---------|
| **Starter** | 注册即用（100 免费积分） | 基础设计生成 |
| **Pro（推荐）** | 登录后可见 | 更多积分 + 高级模型访问 |
| **Premium** | 登录后可见 | 最高积分配额 + 全部功能解锁 |

### 4.2 定价策略特点

- **积分制（Credits）**：按生成次数/复杂度消耗，积分永不过期
- **新用户优惠**：注册即送 100 积分 + 首单 15% OFF
- **阶梯定价**：Starter → Pro → Premium 递进，覆盖不同使用频次用户
- **登录后展示价格**：定价详情需注册登录可见（提升注册转化）

### 4.3 用户激励

- 新用户注册奖励（100 Credits）
- 限时折扣活动（15% OFF）
- 积分永不过期（降低决策门槛）

## 五、竞争格局分析

### 5.1 竞品对比

| 维度 | **Lovart** | **Canva** | **Adobe Express** | **Midjourney** | **Recraft** |
|------|-----------|-----------|------------------|---------------|-------------|
| **定位** | AI Design Agent | 在线设计工具 | 快速设计工具 | AI 图像生成 | AI 设计工具 |
| **交互方式** | 对话式 Agent | 拖拽编辑 | 模板 + 编辑 | Prompt → 图 | Prompt 编辑 |
| **设计链路** | 端到端（Logo→海报→视频） | 单点设计 | 单点设计 | 仅图像 | Logo + 插画 |
| **视频生成** | ✅（Sora/Veo） | ✅（有限） | ❌ | ❌ | ❌ |
| **模型多样性** | 多模型编排 | 自研 | 自研 | 单一 | 自研 |
| **中文支持** | ✅ 优秀 | ❌ 一般 | ❌ 一般 | ❌ | ❌ |
| **免费模式** | ✅ 100 Credits | ✅ 有限免费 | ✅ 有限免费 | ✅ 有限免费 | ✅ 有限免费 |

### 5.2 Lovart 的核心差异化优势

1. **"Agent" 而非 "工具"** — 用户描述需求，AI 自主规划并执行完整设计流程，而非逐个操作
2. **多模型编排** — 不依赖单一模型，按场景自动选择最优模型组合
3. **覆盖全链路** — 从 Logo 到视频广告，一个会话完成品牌全案
4. **中文原生支持** — 对中国用户友好，prompt 理解能力更强
5. **低门槛** — 无需设计技能，自然语言驱动

### 5.3 潜在挑战

- **品牌认知度**：相比 Canva（估值 $400 亿+）仍有差距
- **编辑精细化**：AI 生成后的人工微调能力是否够强
- **定价透明度**：价格需登录后查看，可能影响决策效率
- **生态建设**：缺少插件/模板市场等生态护城河

| 维度 | 评分 (1-5) | 说明 |
|------|-----------|------|
| 产品理念 | ⭐⭐⭐⭐⭐ | AI Design Agent 定位前瞻，差异化明显 |

| 生态建设 | ⭐⭐ | 模板/插件/社区尚未形成 |

### 8.2 关键建议

1. **明确定价策略**：考虑在公开页面展示价格区间，降低用户决策摩擦
2. **强化客制化能力**：AI 生成后的编辑/微调体验是留存关键
3. **内容营销发力**：增加教程/案例内容，提升 SEO 自然流量
4. **企业版布局**：品牌设计自动化是 B 端大客户的刚需场景
5. **建设社区生态**：设计模板/风格分享社区可形成护城河
